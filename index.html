Introduction

Welcome to DDI-PT

2020/10/02 © 2020 Version 1.1

DDI-PT is an easy-to-use web-based tool that takes LINCS signatures of two drugs in pair as inputs, and accurately predicts relevant drug-drug interaction(DDI) types for the input drug pair.

This website contains four parts:

  • Predict DDI, where you can predict potential DDIs of drug pairs using LINCS signatures.
  • Browse Drug, where you can browse targets, emzymes, transporters, carriers of drugs.
  • Browse LINCS, where you can browse LINCS signatures of drugs.
  • Download, where you can download relevant data.
Predict DDI
Architecture

This neural network contained 7 dense layers of 6 feature extraction layers and 1 task layer. The hidden nodes in the feature extraction layers were set from 1700 to 200 hidden nodes with a drop of 300 nodes per layer. The task layer contained 90 nodes. The tanh was chosen as the activation function except for the last layer with the sigmoid activation function. The model was trained by minimizing prediction errors in comparison with the gold standard DDI dataset from DrugBank using cross entropy as loss function and the Adam method for optimization. The learning rate, β1, β2 were set to 0.0001, 0.5, 0.999, respectively. The model was trained up to 100 epochs.

Input

The Library of Integrated Cellular Signatures (LINCS) is an NIH program which funds the generation of perturbational profiles across multiple cell and perturbation types, as well as read-outs, at a massive scale. The study design involves the generation of a compendium of transcriptional expression data from cultured human cells treated with small-molecule and genetic loss/gain of function perturbagens.

The data were processed through a computational system, that converted raw fluorescence intensities into differential gene expression signatures. 5 levels of data were generated at the stages of pre-processing.

This model uses the level 5 signatures collected from MCF7 cell line treated with 10 μM of small molecules after 24 h, which were extracted from LINCS PHASE I and LINCS PHASE II. The level 5 signatures in the space of 978 landmark genes, which were directly measured, were used as feature vectors for drugs to predict DDIs.

1049 approved drugs were involved. This model can only predict DDIs in these 1049 drugs.

Output/Result

The results return as "Known DDI" and "Predicted DDI". "Known DDI" contains the known DDIs in DrugBank of the drug pair. "Predicted DDI" contains the predicted DDIs of the drug pair with their probabilities in descending order.

  • Drug file: Drug information containing 3013 drugs with DrugBank id, name, type, cas number.
  • Protein file: Protein information containing 5684 proteins with DrugBank id, type, name, organism, gene symbol, uniprot id.

  • Drug file: Drug information containing 1049 drugs with DrugBank id, name, cas number, PubChem id.

  • LINCS file: LINCS signature with 1049 drugs and 2660 genes.
